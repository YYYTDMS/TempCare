# Data Sources
In accordance with the requirements of the PhysioNet Clinical Databases, we are unable to openly share the data without prior permission. Researchers interested in accessing the data can submit their requests through the following website, and a concise application process can be found here: [MIMIC-III](https://physionet.org/content/mimiciii/1.4/)

Thanks to [ThunderbornSakana](https://github.com/ThunderbornSakana/PyTorch-Implementation-of-Models-Based-on-Longitudinal-EHR-Data) repo for sharing their data processing.
